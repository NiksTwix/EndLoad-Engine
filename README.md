Игровой движок на базе OpenGL и ECS-архитектуры. Находится в активной разработке.

## 🔥 Особенности
- Entity-Component-System (ECS) архитектура
- Поддержка ассетов через Assimp
- Поддержка шрифтов (FreeType)
